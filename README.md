# RobôCIn Website

Code for the RobôCIn website, built with React + Vite.

## Getting Started

Requires Node.js.

```bash
npm install
npm run dev
```

## Available Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start the development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run ESLint |
